# 그래픽 아트 리스트 앱 UI

## 소개

이 UI는 Figma를 사용하여 디자인되었으며, 그래픽 아트 작품들을 효과적으로 관리하고 시각적으로 탐색할 수 있는 간편한 인터페이스를 제공합니다.

## 사용된 기술

- Figma (UI 디자인 도구)
- Locofy (HTML 변환 도구)
