# 🚀 회사 홈페이지 제작

이 폴더는 회사 홈페이지를 제작하는 과제입니다. 홈페이지에는 회사 소개에 대한 내용이 포함되어있습니다.

## 🛠️ 사용 기술

- HTML
- CSS
- PHP
- JavaScript
- JQ

## 📁 프로젝트 구조

- `index.html`: 홈페이지 메인 파일
- `assets/`: CSS 및 JS 파일들이 있는 폴더
- `scripts/`: JavaScript 파일들이 있는 폴더
- `images/`: 홈페이지에서 사용되는 이미지들
- 기타 필요한 파일 및 폴더 명시
